Quiz 02 Sample [10 marks], time allowed: 20 minutes

Q1
(a) Write a class StringUtility2 with a string attribute phrase. Also write its getter and setter methods. The names of getter and setter methods should be getWord() and setWord() respectively if the attribute name is word.

(b) Add a constructor StringUtility2() without a parameter which assigns the string "default string" to phrase.

(c) Add another constructor StringUtility2() with a string parameter to initialize phrase.

(d) Add a method deleteFirst(String substring) which returns phrase with the first occurrence of substring deleted if phrase contains substring. Otherwise, it returns phrase unchanged.

(e) Add a method countLetter(char aLetter) which returns the number of aLetter in phrase. For example, if phrase contains " count Letter ", the call countLetter('e') will return 2.

After writing your program, please compile and run it using the 'Compile and Run' button. If everything is ok, press the 'Compile and Submit' button to submit it. You can submit more than once. Only the last submission (within the time limit) will be counted.

Hint: you may like to write the main() method to test the correctness of your class before submission.
