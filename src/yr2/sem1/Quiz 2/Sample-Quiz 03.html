Quiz 03 Sample [10 marks], time allowed: 25 minutes
(This is a sample quiz. Marks will not be counted.)

Q1
(a) Write a class ListClass2 with an attribute myList which is an ArrayList of String. Also write the getter and setter methods of the attribute. The names of getter and setter methods should be getWord() and setWord() respectively if the attribute name is word.

(b) Add a method shortest() which returns the length of the shortest string in myList.

(c) Add a method count(String substr) which returns the number of elements in myList containing substr as a part of the element.

(d) Add a method indexOf(String string, int start) which finds the first occurrence (smallest index) of string in myList with index >= start. It returns -1 if the string is not found.

(e) Add a method concatFirstChar() which returns the a string formed by the first character of each element in the list (in ascending index order). For example, if myList = ["abc", "bcd", "def"], the method should return "abd".
